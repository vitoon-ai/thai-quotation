ใช้งานได้ใน Claude Code ทุกที่ — ฟรี โอเพนซอร์ส

1 · เพิ่ม & ติดตั้ง

/plugin marketplace add vitoon-ai/thai-quotation
/plugin install thai-quotation@boombignose-local

จากนั้นรีโหลด session แล้วใช้คำสั่ง /quotation ได้ทันที

2 · ลองสร้างใบแรก

/quotation
# ระบบจะถามข้อมูลลูกค้า → รายการ → ส่วนลด
# ✓ บันทึก quotations/QT-20260605-01.html

ในกล่องมีอะไร

  • ✓ คำสั่ง /quotation (โหมดถาม + โหมด JSON)
  • ✓ เทมเพลต HTML A4 พร้อมพิมพ์ PDF
  • ✓ ไฟล์ตัวอย่าง seller & quotation
  • ✓ เอกสารภาษาไทยครบ (README)
  • ✓ บันทึก .html + .json ทุกใบ ไว้แก้/ออกใหม่

ต้องการเฉพาะ Claude Code — ไม่ต้องลง Python หรือไลบรารีใด ๆ
