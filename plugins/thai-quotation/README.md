# Thai Quotation Plugin — ปลั๊กอินสร้างใบเสนอราคาภาษาไทย

สร้าง **ใบเสนอราคา (Quotation)** ภาษาไทยเป็นไฟล์ HTML ขนาด A4 พร้อมพิมพ์เป็น PDF
คำนวณ **VAT 7%** และ **จำนวนเงินตัวอักษร** ("...บาทถ้วน") ให้อัตโนมัติ
ปรับแต่งข้อมูลบริษัทผู้ขายได้เอง — ใช้ได้กับทุกธุรกิจ (generic)

> เอกสารแบบ **2 ภาษา** (ไทย/อังกฤษ) · วันที่แบบ **พ.ศ.** · เลขที่เอกสารอัตโนมัติ · รองรับ **หัก ณ ที่จ่าย** (เปิด/ปิดได้)

---

## การติดตั้ง (Local Marketplace)

1. เพิ่ม marketplace เข้าระบบ:
   ```
   /plugin marketplace add d:\ClaudeCode-Workshop\marketplace
   ```
2. ติดตั้งปลั๊กอิน:
   ```
   /plugin install thai-quotation@boombignose-local
   ```
3. รีโหลด / เริ่ม session ใหม่ เพื่อให้คำสั่ง `/quotation` ใช้งานได้

---

## ตั้งค่าครั้งแรก — ข้อมูลผู้ขาย (seller.json)

ปลั๊กอินอ่านข้อมูลบริษัทผู้ขายจากไฟล์ (อยู่นอกตัวปลั๊กอิน เพราะปลั๊กอินเป็นแบบอ่านอย่างเดียว):

```
d:\ClaudeCode-Workshop\.claude\quotation\seller.json
```

ครั้งแรกให้คัดลอกไฟล์ตัวอย่างมาวางแล้วแก้ข้อมูลของคุณ:

```
สำเนา:  marketplace\plugins\thai-quotation\examples\seller.example.json
ไปที่:  .claude\quotation\seller.json
```

ฟิลด์ใน `seller.json`:

| ฟิลด์ | ความหมาย | จำเป็น |
|---|---|:--:|
| `companyName` | ชื่อบริษัทผู้ขาย | ✅ |
| `address` | ที่อยู่ | ✅ |
| `taxId` | เลขประจำตัวผู้เสียภาษี | ✅ |
| `phone` | เบอร์โทร | ✅ |
| `email` | อีเมล | – |
| `logoPath` | พาธรูปโลโก้ (เว้นว่างได้) | – |
| `vatRate` | อัตรา VAT (ค่าเริ่มต้น `7`) | ✅ |
| `priceMode` | ต้องเป็น `"exclusive"` = ราคาที่กรอกยังไม่รวม VAT | ✅ |
| `withholdingTax` | `{ "enabled": false, "rate": 3 }` เปิด/ปิด หัก ณ ที่จ่าย | – |
| `validDays` | จำนวนวันที่ใบเสนอราคามีผล (ค่าเริ่มต้น `30`) | – |
| `paymentTerms` | เงื่อนไขการชำระเงิน | – |
| `preparedBy` | ชื่อผู้จัดทำ/ผู้เสนอราคา | – |

## วิธีใช้งาน

```
/quotation                          # โหมดถามทีละข้อ (Guided)
/quotation { ...JSON... }           # วาง JSON ข้อมูลใบเสนอราคามาตรง ๆ
/quotation path\to\quotation.json   # อ่านจากไฟล์ JSON
```

### โหมดถามทีละข้อ (Guided)
ปล่อย `/quotation` ว่าง ระบบจะถามข้อมูลลูกค้า → รายการสินค้า/บริการ → ส่วนลด → หมายเหตุ ทีละขั้น

### โหมดป้อนข้อมูล (JSON)
รูปแบบข้อมูล (ดูตัวอย่างเต็มที่ `examples\quotation.example.json`):

```json
{
  "customer": {
    "companyName": "บริษัท ลูกค้า จำกัด",
    "address": "456 ถนน... กรุงเทพฯ 10110",
    "contactName": "คุณสมชาย ใจดี",
    "phone": "02-111-2222",
    "taxId": "0-1234-56789-00-1"
  },
  "items": [
    { "description": "พัฒนาเว็บไซต์บริษัท", "qty": 1, "unit": "โปรเจค", "unitPrice": 45000 },
    { "description": "เว็บโฮสติ้ง", "qty": 12, "unit": "เดือน", "unitPrice": 2500 }
  ],
  "discount": 0,
  "note": "ราคานี้ยังไม่รวมค่าโฆษณา"
}
```

ฟิลด์ต่อรายการที่ต้องมี: `description`, `qty`, `unit`, `unitPrice`
(`discount` และ `note` ใส่หรือไม่ก็ได้ — ไม่ใส่ = 0 / ว่าง)

## ผลลัพธ์

ระบบจะบันทึก 2 ไฟล์ที่ `d:\ClaudeCode-Workshop\quotations\`:

| ไฟล์ | ใช้ทำอะไร |
|---|---|
| `QT-YYYYMMDD-NN.html` | เอกสารใบเสนอราคา (เปิดในเบราว์เซอร์) |
| `QT-YYYYMMDD-NN.json` | ข้อมูล + ยอดที่คำนวณแล้ว (ไว้แก้ไข/ออกใหม่) |

เลขที่เอกสารรันอัตโนมัติต่อวัน เช่น ใบแรกของวัน = `QT-20260605-01`, ใบถัดไป = `-02`

### ส่งออกเป็น PDF
เปิดไฟล์ `.html` ในเบราว์เซอร์ → กด **Ctrl/Cmd + P** → เลือก **Save as PDF** → ขนาด **A4**

## การคำนวณ

- ราคาที่กรอกเป็นราคา **ก่อน VAT** (`priceMode: "exclusive"`)
- `รวมเป็นเงิน` = Σ (จำนวน × ราคาต่อหน่วย)
- หักส่วนลด (ถ้ามี) → `ภาษีมูลค่าเพิ่ม 7%` → `ยอดรวมทั้งสิ้น`
- ถ้าเปิด `withholdingTax` จะแสดง `หัก ณ ที่จ่าย` และ `ยอดชำระสุทธิ` เพิ่ม
- แปลง `ยอดรวมทั้งสิ้น` เป็นจำนวนเงินตัวอักษรภาษาไทยให้อัตโนมัติ

---

## ขอบเขต (Scope)

- ทำเฉพาะ **ใบเสนอราคา** (ไม่รวมใบแจ้งหนี้/ใบเสร็จ)
- สกุลเงิน **บาท (THB)** เท่านั้น
- ราคาที่กรอกเป็นราคา **ก่อน VAT** (รองรับเฉพาะ `priceMode: "exclusive"`)
- ส่งออก PDF ผ่านเบราว์เซอร์ (ไม่พึ่ง Python หรือไลบรารีภายนอก)
